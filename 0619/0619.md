## Scalar 연산
- Arithmetic 연산 (Numeric 데이터의 연산)
    - 사칙 연산: + - * /
    - %%: a %% b == a를 b로 나눈 나머지
    - %/%: a %/% b == a를 b로 나눈 몫

- Logical 연산
    - not !
        - not True == False
        - not False == True
    - and & , 교집합 *
        - True and True == True
        - True and False == False
        - False and True == False
        - False and False == False
    - or | , 합집합 +
        - True or True == T
        - True or False == T
        - False or True == T
        - False or False == F

- 비교 연산
    - 같냐? ==
    - 다르냐? !=
    - >
    - <
    - >= (=> X)
    - <=
