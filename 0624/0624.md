## Scalar
- Scalar 데이터의 종류
    - Numeric
        - Integer: 1L
        - Double: 1 1.0
    - Character
        - 'abc' "abc"
    - Logical
        - T, TRUE
        - F, FALSE
    - NA
        - Not available
    - NULL
        - Undefined
- `typeof()`
    - 어떤 종류의 scalar인지 알 수 있다.
    - 예)
        ```R
        > typeof(1L)
        [1] "integer"
        ```
- `is.데이터타입()`
    - 특정 데이터 타입인지 아닌지 알 수 있다.
    - 예)
        ```R
        > is.integer(1L)
        [1] TRUE
        ```
- Scalar 연산
    - Numeric 연산
        - 사칙연산: +, -, \*, /
        - 몫: %/%
            - `a %/% b` == a를 b로 나눈 몫
            - 예)
                ```R
                # 20을 3으로 나눈 몫은 6
                > 20 %/% 3
                [1] 6
                ```
        - 나머지: %%
            - `a %% b` == a를 b로 나눈 나머지
            - 예)
                ```R
                # 20을 3으로 나눈 나머지는 2
                > 20 %/% 3
                [1] 2
                ```
        - 거듭제곱: ^ \*\*
    - 비교 연산
        - == 같은지
        - != 다른지
        - \> 큰지
        - < 작은지
        - \>= 크거나 같은지
        - <= 작거나 같은지
    - Character 연산
        - `nchar()`
            - number of character?
            - 문자열의 길이
            - 예)
                ```R
                > nchar('Love')
                [1] 4
                ```
        - `paste(문자열1, 문자열2, ..., sep=세퍼레이터)`
            - 여러 문자열 붙이기
            - 예)
                ```R
                > paste('Love', 'is', 'an', 'open', 'door!', sep=' ')
                [1] "Love is an open door!"
                ```
        - substr('abcd', 2, 3)
    - Logical 연산
        - !
        - &
        - |
